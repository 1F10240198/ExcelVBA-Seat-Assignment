# ExcelVBA-Seat-Assignment
# 講師座席自動配置システム(Excel VBA)

## 概要と開発経緯
塾全体の時間割変更や、勤務教室独自の方針（分室の存在、自習席の増設）に柔軟に対応するために作成しました。
# 主な変更
時間割変更に伴う問題
・旧システムでは判定していないコマが発生した。（根本から変更する必要）
・休憩時間を講師側が自分で確保する必要があり、休憩なしでの勤務を防止する必要がある。
自習室の増設
・通塾している生徒からの分室も自習室として利用したいという意見。
・自習室利用のために分室に講師を常に置く必要がある。

## 工夫・こだわり
処理を機能ごとに関数化することで、将来のルール変更や後輩への引継ぎを容易にする。
生徒の個人情報や機密性の高い授業情報をシステム内の検索条件から完全に排除し、安全に運用できる設計を徹底した。
